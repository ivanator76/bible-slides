# 聖經經文投影片產生器 Bible Verse Slide Generator

一鍵查詢聖經經文，自動生成精美 PowerPoint 投影片。

## 部署到 GitHub Pages

### 步驟一：建立 GitHub Repository

1. 登入 [github.com](https://github.com)
2. 點擊右上角 **+** → **New repository**
3. Repository name 輸入：`bible-slides`（或任何你喜歡的名稱）
4. 選擇 **Public**
5. 點擊 **Create repository**

### 步驟二：上傳檔案

1. 在新建的 repository 頁面，點擊 **Add file** → **Upload files**
2. 將 `index.html` 和 `README.md` 拖入上傳區
3. 在下方 Commit 訊息輸入：`Add bible slide generator`
4. 點擊 **Commit changes**

### 步驟三：啟用 GitHub Pages

1. 進入 repository 的 **Settings**（上方選單）
2. 左側選單找到 **Pages**
3. Source 選擇 **Deploy from a branch**
4. Branch 選擇 **main**，資料夾選 **/ (root)**
5. 點擊 **Save**
6. 等待 1-2 分鐘，頁面頂部會出現網址

### 步驟四：取得網址

部署完成後，你的網址會是：

```
https://你的帳號.github.io/bible-slides/
```

在手機或任何裝置上用這個網址開啟即可使用。
